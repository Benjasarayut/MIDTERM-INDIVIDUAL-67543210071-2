Library Management System

Server is running. Use the API endpoints under /api/books.

  • GET /api/books
  • GET /api/books/:id
  • POST /api/books (JSON)
  • PUT /api/books/:id (JSON)
  • PATCH /api/books/:id/borrow
  • PATCH /api/books/:id/return
  • DELETE /api/books/:id

If you still see "Cannot GET /" in the browser, try clearing the browser cache or open the exact URL /.
